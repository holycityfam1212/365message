[기획안] 2026 온라인 신년 말씀 뽑기 웹 페이지

1. 프로젝트 개요
프로젝트명 : 2026 내게 주시는 하나님의 말씀 (부제: 말씀대로 살아내는 한 해)
기획 의도 : 말씀을 뽑고 그 말씀대로 살아보는 디지털 경험 제공
sns 공유 기능을 활용한 ‘오늘의 메시지’ 채널 활성화
가챠 형식을 빌린 재미 요소를 통해 전 연령층 참여 유도
타겟 사용자 : 스마트폰이 사용 가능한 전 연령층(교회 성도, 청년, sns이용자 등)
플랫폼 : 모바일 웹 중심

2. 핵심 목표 및 기대 효과
접근성 : 별도의 회원가입 없이 ‘원클릭’으로 즉시 참여 가능한 경험 제공
몰입감 : 실제 가챠 기계를 돌리는 듯한 시청각 애니메이션으로 재미 극대화
확산성 : 결과 이미지 저장 및 카카오톡/ 인스타그램 공유를 통한 자발적 콘텐츠 확산

3. 상세 기능
A. 사용자 인터랙션 흐름 (User Flow)
1) 랜딩 페이지
- ‘오늘의 메시지’브랜드 로고와 2026 타이틀 배치
- 정면 가챠 기계 일러스트와 함께 “동전을 넣고 뽑아 보세요” 안내 문구 노출

2) 뽑기 액션
- 동전 투입 : 사용자가 동전 오브젝트를 기계 투입구로 드래그 혹은 클릭
애니메이션 : 핸들이 돌아가며 기계 안의 캡슐들이 섞이는 효과 연출 (약 1.5초)
결과 배출 : 캡슐 하나가 배출구로 떨어지고, 
     사용자가 클릭하면 캡술이 열리며 말씀 카드 등장

3) 결과 페이지
랜덤 추출된 성경 구절(본문, 장절) 표시
[이미지 저장]버튼 : 말씀 카드를 이미지 파일로 다운로드
[공유하기]버튼 : 카카오톡 API 연동 및 링크 복사 기능
[다시 뽑기]버튼 : 무제한 허용하되, ‘하루 한 번, 뽑은 말씀대로 살아보기’라는 메시지 하단에 노출

B. 데이터 구성 및 관리
말씀 데이터 베이스 : 총 130개의 성경 구절을 JSON 형식으로 관리(ID, 본문, 장절, 테마)
테마 분류 : 축복과 소망/ 위로와 평안/ 능력과 용기/ 지혜와 인도/ 사랑과 섬김/ 믿음과 헌신
(각 테마별 배경 디자인 랜덤 매칭)

4. 디자인 컨셉
웹 환경 : 모바일 웹 중심의 반응형 레이아웃 작업물
가챠 머신 소스
기계 본체/ 회전 가능한 핸들 파츠/ 알록달록한 원형 캡슐/ 동전 오브젝트
톤 앤 매너 : 블루와 따뜻한 파스텔 톤을 사용
디자인 요소 : 말씀 카드 이미지 첨부(말씀 폰트 : NotoSerifCJKKR / 자간 –80/ 행간 15)
폰트 : 가독성이 좋은 명조체

5. 에셋 리스트
그래픽
SNS 공유 섬네일 : (1200*630) 카카오톡 또는 인스타그램 링크 공유용
파비콘 : 32*32 웹브라우저 탭 표시용 아이콘
가챠 기계 본체 : 내부가 투명하게 보이는 정면 기계 일러스트
가챠 핸들 : 애니메이션 구현을 위해 기계 본체와 분리된 핸들 파츠
투입용 동전 : 사용자가 드래그할 입체적인 코인
가챠용 캡슐 : 기계 안에서 섞일 다양한 색상의 원형 캡슐
배출된 캡슐 : 배출구로 나온 캡슐 및 개봉(반으로 갈라진) 상태의 소스
말씀카드 : 말씀이 적혀 있는 이미지 파일(저장용 파일)

데이터 콘텐츠
성경구절 DB (공유)
말씀 폰트 : NotoSerifCJKKR / 자간 –80/ 행간 15
